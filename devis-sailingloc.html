DEVIS - PROJET SAILINGLOC

Date : 07/08/2025

Référence : DEV-SL-2025-001

Client : SailingLoc

1. PRÉSENTATION DU PROJET

Développement d'une application web de location de bateaux (voiliers et bateaux à moteur) permettant aux utilisateurs de rechercher, consulter et réserver des bateaux en ligne.

2. FONCTIONNALITÉS DÉVELOPPÉES

2.1 Interface utilisateur

  • Page d'accueil avec recherche par localisation et date
  • Navigation entre les différentes catégories de bateaux (voiliers/bateaux à moteur)
  • Affichage des destinations populaires
  • Témoignages d'utilisateurs
  • Design responsive et moderne

2.2 Catalogue de bateaux

  • Liste des voiliers disponibles avec filtres (prix, capacité)
  • Liste des bateaux à moteur disponibles avec filtres
  • Fiches détaillées des bateaux avec caractéristiques techniques
  • Galerie d'images pour chaque bateau
  • Système de favoris

2.3 Système de réservation

  • Sélection des dates de location
  • Calcul automatique du prix en fonction de la durée
  • Formulaire de réservation
  • Système de paiement en ligne

2.4 Fonctionnalités utilisateur

  • Création et gestion de compte
  • Historique des réservations
  • Système d'avis et de notation

2.5 Administration

  • Gestion du catalogue de bateaux
  • Gestion des réservations
  • Tableau de bord statistique

3. TECHNOLOGIES UTILISÉES

  • Frontend : React 19, React Router 7, Bootstrap 5, Tailwind CSS
  • Bibliothèques UI : FontAwesome, Headless UI
  • Cartographie : React Google Maps API
  • Authentification : JWT
  • Build et déploiement : Vite 6
  • Tests : Vitest, React Testing Library
