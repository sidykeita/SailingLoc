5. COMPOSANTS TRANSVERSAUX

5.1 Header

Éléments du header

  • Logo : SailingLoc (150x50px, position gauche)
  • Navigation : Menu horizontal centré
  • Actions : Connexion, Inscription, Panier (position droite)
  • Responsive : Menu burger sur mobile

5.2 Footer

Éléments du footer

  • Informations légales : Mentions légales, CGV, Confidentialité
  • Contact : Adresse, téléphone, email
  • Réseaux sociaux : Facebook, Instagram, Twitter
  • Newsletter : Formulaire d'inscription

5.3 Cartes bateau

Éléments des cartes

  • Image : Photo principale (350x200px)
  • Titre : Nom du bateau
  • Localisation : Port d'attache
  • Caractéristiques : Longueur, capacité, puissance
  • Prix : Prix par jour
  • Actions : Bouton favoris, lien détails

5.4 Formulaires

Éléments des formulaires

  • Champs de saisie : Style uniforme avec bordures arrondies
  • Boutons : Couleur principale #274991, hover #66C7C7
  • Validation : Messages d'erreur en rouge, succès en vert
  • Responsive : Adaptation mobile avec champs pleine largeur

6. SPÉCIFICATIONS TECHNIQUES

6.1 Breakpoints responsive

Points de rupture

  • Mobile : < 768px (1 colonne)
  • Tablet : 768px - 1024px (2 colonnes)
  • Desktop : > 1024px (3 colonnes)
  • Large Desktop : > 1200px (4 colonnes max)

6.2 Charte graphique

Couleurs principales

  • Bleu principal : #274991
  • Turquoise : #66C7C7
  • Gris foncé : #333333
  • Gris clair : #f8f9fa
  • Blanc : #ffffff

Typographie

  • Police principale : Roboto (Google Fonts)
  • Titres H1 : 48px Bold
  • Titres H2 : 36px Bold
  • Titres H3 : 24px Medium
  • Texte courant : 16px Regular

6.3 Animations et interactions

Effets visuels

  • Hover cartes : Élévation avec ombre portée
  • Boutons : Transition couleur 0.3s
  • Images : Effet zoom léger au survol
  • Scroll : Bouton retour en haut après 300px

7. FLUX UTILISATEUR

7.1 Parcours de réservation

Étapes principales

  • 1. Recherche : Saisie localisation et dates
  • 2. Résultats : Liste des bateaux disponibles
  • 3. Sélection : Choix d'un bateau et consultation détails
  • 4. Réservation : Sélection dates précises et options
  • 5. Identification : Connexion ou création compte
  • 6. Paiement : Saisie informations et validation
  • 7. Confirmation : Email et récapitulatif réservation

7.2 Gestion compte utilisateur

Fonctionnalités compte

  • Profil : Informations personnelles et préférences
  • Réservations : Historique et réservations en cours
  • Favoris : Bateaux sauvegardés
  • Messages : Communication avec propriétaires
  • Évaluations : Avis laissés et reçus
